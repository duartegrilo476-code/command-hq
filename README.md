# Command HQ Dashboard

Dashboard de gestão para Crypto & Forex com agentes IA.

## Como usar

1. Abre o site
2. Clica em 🔑 Gemini API no topo
3. Cola a tua API key gratuita do Google AI Studio
4. Usa os agentes!

## API Key Gemini (gratuita)

Obtém em: aistudio.google.com → Get API key → Create API key
